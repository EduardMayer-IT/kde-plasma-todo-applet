# Meine Aufgabenliste

Ein natives To-Do-List-Applet für KDE Plasma 6.

## Funktionen

- Hinzufügen, Bearbeiten und Löschen von Aufgaben
- Prioritätsstufen für Aufgaben
- Fälligkeitsdaten für Aufgaben
- Synchronisation mit Nextcloud
- Drag & Drop-Unterstützung
